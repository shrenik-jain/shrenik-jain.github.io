# Shrenik Jain — Digital Space


Most of what I build never gets a bookmark-friendly URL — it lives in notebooks, training jobs, and dashboards behind auth. This site is the exception: a hand-typed, fast-loading front door where I line up projects, a work timeline, and enough narrative that “machine learning engineer” reads like a person, not a keyword cluster. The live website is the version I show someone over coffee.

**Live Website:** [shrenik-jain.github.io](https://shrenik-jain.github.io)

<br>

## About the Repository

- Hand-maintained **HTML, CSS, and JavaScript** pages (no build step required to browse the source).
- Published with **GitHub Pages** from the default branch.
- Styling builds on the **PhotoFolio** Bootstrap template, with custom CSS and layout changes throughout.

## Explore the Website

- **Home** — introduction and entry points to the rest of the site  
- **Work** — employment, research, and consulting timeline  
- **Projects** — selected work with links to repositories  
- **Skills** — programming languages, ML stack, tooling, and platforms  
- **Academics** — education and certifications  
- **Testimonials** — recommendations from collaborators  
- **Contact** — form, Calendly, email, and phone
